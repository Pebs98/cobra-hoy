# Cobra HOY

**Structured finance solution for healthcare receivables — SEOGA x Laminar**

Cobra HOY enables hospitals to receive early payment on expected insurer payouts, using SEOGA's granular data intelligence on medical activity and insurer payment behavior.

## Repository structure

```
docs/                        Source documents (markdown)
  master-architecture.md     Master architecture & components overview
  engineering/               Technical specs and pipeline design
  internal/                  Internal memos and strategy docs
  investor/                  Executive summaries and investor materials
  legal/                     Legal counsel memos and questions
assets/
  diagrams/                  Interactive HTML diagrams and mermaid sources
  deliverables/              Standalone deliverables (two-pager, screenshots)
reference/                   Background reference material
index.html                   Live web page (GitHub Pages)
```

## Live document

The master architecture document is published via GitHub Pages at the repository's Pages URL. It renders `docs/master-architecture.md` live — edits to the markdown are reflected on the next page load.
